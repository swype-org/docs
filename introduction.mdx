## Welcome to Blink

**Blink is a deposit SDK for crypto apps.** It lets your users deposit crypto into your app with passkeys, instead of dealing with the friction of addresses, chains, or tokens.

### It takes about 5 minutes to add

Drop in one button next to your existing payment methods. We handle the hard parts:

- No wallet UI to build
- No on-chain logic to write
- Nothing to PCI-scope

Behind the button, Blink handles authentication, wallet connection, on-chain transfers, and cross-chain bridging.

### Best for apps with embedded wallets

Prediction markets, trading apps, games, and AI agents. If your users need to get money in fast, Blink is the funding layer that gets them in.

### Blink is not a wallet

We work on top of wallet infrastructure like Privy, Turnkey, and Dynamic, and help your users fund those wallets in one tap. You don't replace anything you already have.

### Web and mobile

Both SDKs are available. Web renders the deposit flow in a modal iframe. Mobile opens the same flow in an in-app browser and returns the result through a deep link.

For the current wallets, chains, tokens, and fees, see [Supported Networks and Wallets](/integration/supported-networks-and-wallets).
