## Welcome to Blink!

Blink is a hosted deposit flow for crypto apps. Your app opens a secure modal iframe powered by Blink, and users complete crypto and fiat deposits in seconds without ever leaving your site.

There is nothing to PCI-scope, no wallet UI to build, and no on-chain logic to manage. Blink handles user authentication, wallet connection, on-chain transfers, and cross-chain bridging.

Both **web** and **mobile** SDKs are available. The web SDK renders the payment flow in a modal iframe overlay. The mobile SDK opens the same hosted flow in an in-app browser and returns the result via a deep link callback.

These docs cover everything from generating your key pair to shipping a production deposit integration.
